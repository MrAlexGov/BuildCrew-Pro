# BuildCrew Pro - SaaS для управления строительными бригадами

## Описание

**BuildCrew Pro** - это облачное SaaS-решение для автоматизации управления строительными бригадами, объектами и задачами. Продукт повышает производительность, прозрачность и контроль над строительными проектами для прорабов, руководителей и владельцев строительного бизнеса.

## Архитектура проекта

### Технический стек
- **Бэкенд**: Node.js (NestJS) - TypeScript
- **Фронтенд**: React/TypeScript
- **База данных**: PostgreSQL
- **Файловое хранилище**: AWS S3 (или аналоги)
- **Контейнеризация**: Docker
- **Деплой**: Облачные платформы (Yandex Cloud, AWS, Selectel)

### Модули системы

#### 1. Панель управления (Dashboard)
- Сводка по активным объектам
- Ход выполнения работ
- Освоенный бюджет
- Просроченные задачи

#### 2. Управление объектами
- Создание и редактирование объектов
- Управление информацией по объектам
- Назначение ответственных прорабов

#### 3. Управление бригадами и сотрудниками
- Создание профилей сотрудников
- Формирование бригад
- Календарь занятости

#### 4. Планирование и постановка задач
- Создание задач с назначением
- Использование шаблонов типовых работ
- Контроль сроков и приоритетов

#### 5. Отчетность и учет рабочего времени
- Статусы выполнения задач
- Фотоотчетность
- Табель учета рабочего времени

#### 6. Фотоотчетность и документация
- Прикрепление фото к задачам
- Облачное хранилище документов
- Галерея объекта

#### 7. Уведомления и коммуникация
- Push-уведомления
- Система оповещений о проблемах

#### 8. Аналитика и отчеты
- Трудозатраты по объектам/бригадам
- Освоение бюджета
- Продуктивность бригад
- KPI прорабов

### Ролевая модель
- **Владелец** - полный доступ ко всем функциям
- **Руководитель** - управление несколькими объектами
- **Прораб** - управление своими объектами и бригадами
- **Рабочий** - выполнение задач и отчетность

## Запуск проекта

### Требования
- Node.js 18+
- PostgreSQL
- Docker (опционально)

### Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```
3. Настройте окружение (см. `.env.example`)
4. Запустите разработку:
   ```bash
   # Backend
   npm run start:dev
   
   # Frontend
   npm run dev
   ```
